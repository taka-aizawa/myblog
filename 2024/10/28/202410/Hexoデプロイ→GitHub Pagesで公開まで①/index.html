Hexoデプロイ→GitHub Pagesで公開まで①

学んだ知識のメモ書きとして、技術ブログの立ち上げを思い立つ。

QiitaやZennなどのを既存ツールを検討したものの、自由度や視認性の高さなどに惹かれて静的サイトジェネレーターに決めた。テーマはHexoのicarus。

当初はNetlify+GitHubなんて流行りっぽい構成で作ろうと思ったが（公開を制限できるのもメリットだった）、安定感も大事だと思い直して最終的にGithub Pagesに落ち着いた。

GitHub Pagesは無料だとパブリックしか選択できないのが気になるところ。
まあ、気になったら後々GitHub PagesからNetlifyにも移行できるので。
